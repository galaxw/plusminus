# 项目功能需求

## 1. 技术栈
- 使用 C++ 进行开发。

## 2. 功能定位
- 开发一个面向普通用户视频剪辑软件。

## 3. 交互方式
- 支持图形用户界面 (UI) 和命令行 (CLI) 两种方式完成视频剪辑。

## 4. 素材支持
- 图片：JPEG、PNG、BMP、GIF 等。
- 视频：MP4、AVI、MKV、MOV、WMV 等。
- 音频：MP3、WAV、AAC、FLAC、OGG 等。

## 5. 输出格式
- 支持 MP4、AVI、MKV、MOV、WMV 等常见视频格式。
- 支持 H.264、H.265、VP9 等编码标准。
- 允许自定义分辨率、帧率、比特率等参数。

## 6. 剪辑特效
- 转场效果：淡入淡出、滑动、覆盖等。
- 滤镜效果：灰度、黑白、复古、锐化、模糊等。
- 文字叠加：字体、颜色、位置、动画。
- 动态贴纸/表情：GIF 或静态图片叠加。
- 音视频同步调整：音量调节、音频淡入淡出。
- 关键帧动画：实现复杂特效变化。

## 7. AI 素材生成
- 支持将素材发送到大模型 API 进行 AI 处理。
- 将生成内容作为剪辑素材集成到项目中。

## 8. 项目管理
- 以项目为单位进行视频剪辑，包含素材、时间线、特效设置、导出配置等。
- 支持项目保存、加载、重命名、删除等操作。
- 支持多项目切换。
- 提供项目文件格式（如 `.plusminus`）。

## 9. 命令行模式
- 支持 CLI 剪辑操作。
- 提供指令如 `plusminus-cli edit`、`plusminus-cli render`。
- CLI 与 UI 功能保持一致。

## 10. 视频对象处理
- 支持特定对象消除（如水印、广告等）。
- 支持特定对象替换（如车牌、物体替换）。
- 整合 AI 模型用于对象识别和处理。

## 11. 目标平台
- macOS 和 Windows，暂不支持 Linux。

## 12. 依赖库
- FFmpeg：用于视频编解码、转码、剪辑等功能。
- OpenCV：用于图像处理、AI 对象检测与替换。
- Qt：用于构建跨平台的图形用户界面。

## 13. 语言支持
- 中文界面。
