# 模块划分与功能描述

## 1. UI 模块
- 提供图形用户界面（Qt 框架）
- 支持视频剪辑操作、项目管理、特效设置等
- 支持中文界面

## 2. CLI 模块
- 提供命令行接口（`plusminus-cli`）
- 实现视频剪辑与导出功能
- 与 UI 模块功能保持一致，但不实现全部 UI 能力

## 3. 项目管理模块
- 支持项目创建、加载、保存、重命名、删除
- 管理素材、时间线、特效配置、导出设置等
- 支持多项目切换

## 4. 素材处理模块
- 支持图片、视频、音频的导入与解析
- 支持多种格式的读取和预览

## 5. 视频剪辑模块
- 支持视频切割、拼接、转场效果
- 支持关键帧动画
- 整合 FFmpeg 进行视频编辑

## 6. 特效处理模块
- 支持滤镜、文字叠加、动态贴纸等特效
- 集成 OpenCV 进行图像处理
- 支持 AI 对象消除/替换

## 7. AI 接口模块
- 提供与大模型 API 的交互接口
- 支持素材发送与生成内容集成

## 8. 输出模块
- 支持视频导出为多种格式（MP4、AVI 等）
- 支持编码参数设置（H.264、H.265 等）
- 整合 FFmpeg 进行视频渲染

## 9. 依赖管理模块
- 使用 Conan 管理第三方库（FFmpeg、OpenCV、Qt）
- 提供构建脚本（CMake）
