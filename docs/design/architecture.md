# 系统架构设计

## 1. 系统总体架构

本视频剪辑软件采用分层模块化设计，主要包括以下核心模块：

- **UI 模块**：提供图形用户界面，支持视频剪辑操作、项目管理、特效设置等。
- **CLI 模块**：提供命令行接口，支持通过命令完成视频剪辑与导出。
- **项目管理模块**：管理项目文件，支持项目创建、加载、保存、重命名、删除等。
- **素材处理模块**：支持图片、视频、音频的导入与解析。
- **视频剪辑模块**：支持视频切割、拼接、转场效果、关键帧动画等。
- **特效处理模块**：支持滤镜、文字叠加、动态贴纸等特效，集成 OpenCV 进行图像处理。
- **AI 接口模块**：提供与大模型 API 的交互接口，支持素材发送与生成内容集成。
- **输出模块**：支持视频导出为多种格式，整合 FFmpeg 进行渲染。
- **依赖管理模块**：使用 Conan 管理第三方库（FFmpeg、OpenCV、Qt）。

## 2. 技术栈
- 开发语言：C++
- UI 框架：Qt
- 视频处理：FFmpeg
- 图像处理：OpenCV
- 依赖管理：Conan
- 构建系统：CMake

## 3. 交互方式
- 支持图形用户界面 (UI) 和命令行 (CLI) 两种方式。
- UI 和 CLI 功能保持一致，确保用户体验统一。
